## 4. Kinerja Proyek

### 4.1 Keberhasilan
1. **Implementasi Teknologi Terbaru**: Berhasil menggunakan Laravel 11 dan Bootstrap 5 dengan performa loading < 3 detik.
2. **Kepatuhan Standar**: Kode program mengikuti standar *Clean Code* dan arsitektur *3-Tier* yang *scalable*.
3. **Kolaborasi Tim**: Sinergi antar peran (PM, SM, Creative, Developer) melalui metodologi Scrum berjalan sangat efektif.
4. **Keamanan**: Implementasi proteksi CSRF, XSS, dan SQL Injection secara default menjamin keamanan data pengguna.

### 4.2 Tantangan & Pelajaran yang Didapat
* **Tantangan**: Kompleksitas UI untuk fitur *tracking manual* agar tetap intuitif bagi pengguna.
* **Risiko yang Dihadapi**: Potensi konflik kode (Merge Conflict) saat integrasi fitur.
* **Pelajaran Penting**: Penggunaan Gitflow secara ketat dan komunikasi harian melalui *Standup Meeting* sangat krusial dalam menjaga integritas kode dan timeline.
